# Credit_Card_Financial_Dashboard
💳 Credit_card_transactions_customer_analysis
💳 Credit Card Transaction Analysis Dashboard

📌 Project Overview

This project focuses on analyzing credit card transaction data to uncover key business insights related to revenue performance, customer behavior, and card usage trends.

Using SQL for data extraction and Power BI for visualization, the project builds an interactive dashboard that helps understand transaction patterns, customer segmentation, and financial performance metrics.

The objective is to demonstrate data analytics, business intelligence, and visualization skills commonly used in data analyst roles.

🎯 Business Problem

Financial institutions need to monitor credit card usage patterns, revenue growth, and customer engagement to make informed business decisions.

This project helps answer questions such as:

Which customer segments generate the most revenue?

What are the top performing card categories?

How is revenue trending over time?

Which states contribute the most transactions?

What is the card activation rate and delinquency risk?
